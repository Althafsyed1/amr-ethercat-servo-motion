# AMR EtherCAT Servo Motion

Small motion-planning and servo-simulation project for Advanced Metal Research bench-servo testing.

Current scope:
- One-axis servo simulator
- Simple position motion command
- Basic motion-planning structure
- Future EtherCAT backend for A6-EC StepperOnline servos

Next step:
Add Python code for simulated servo enable, move command, position update, and target reached status.
